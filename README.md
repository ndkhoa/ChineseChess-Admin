# ChineseChess-Admin

An admin panel for managing ChineseChess data

## Using Bundler 2

### Update Rubygems
```
  gem update --system
```

### Update bundler
```
  gem install bundler
```

### Update Gemfile.lock
```
  bundler update --bundler
```
